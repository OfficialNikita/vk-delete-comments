# Массовое удаление своих комментариев из ВКонтакте

Приложение с графическим интерфейсом для автоматического удаления ваших комментариев из ВКонтакте, используя данные из скачанного архива активности.

## 🚀 Быстрый старт

> Перейдите на страницу [релизов](../../releases) и скачайте `.exe`. Он собирается автоматически через **GitHub Actions** из открытого исходного кода.

Или запустите из исходников:

```bash
pip install -r requirements.txt
python main.py
```

## 🔧 Функционал

- Сканирует HTML/HTM/TXT-файлы из папки с архивом ВК
- Извлекает ссылки на комментарии (вида `https://vk.com/wall-12345_12345?reply=111213`)
- Парсит `owner_id` и `comment_id`
- Отправляет запросы на удаление через VK API
- Обрабатывает ошибки (ограничения, частые запросы, Captcha и т.д.)
- Поддерживает разные кодировки файлов (UTF-8, CP1251 и др.)
- **Автоматическое получение токена** через встроенный браузер — не нужно копировать из DevTools
- Настройки через диалоговое окно в приложении

## 🖥️ Использование

### 1. Скачайте архив данных ВК

Перейдите в настройки [приватности](https://vk.com/data_protection?section=rules&scroll_to_archive=1) → «Запросить копию моих данных» → выберите «Архив активности» → скачайте ZIP-файл.

<img width="452" height="210" alt="image" src="https://github.com/user-attachments/assets/9f49949f-2e4b-4d13-9d2f-92d5ddb74f7d" />

### 2. Извлеките папку comments

Найдите в архиве папку `comments` (содержит .html, .htm или .txt файлы).

<img width="646" height="432" alt="image" src="https://github.com/user-attachments/assets/bd8e9179-e9d8-42e3-ab29-a0a87029f5c1" />

### 3. Запустите приложение

### 4. Используйте интерфейс:

1. **Выберите папку** с файлами комментариев через кнопку «Обзор…»
2. **Нажмите «Войти в VK»** — откроется встроенный браузер
3. **Войдите в свой аккаунт** в браузере
4. **Токен будет получен автоматически** из запросов к VK API
5. **Нажмите «Удалить комментарии»** для запуска процесса

> Токен обновляется автоматически при каждом запросе VK API. Можно обновить вручную кнопкой «Обновить токен».
>
> ⚠️ **Не закрывайте окно браузера** — токен перестанет обновляться, и удаление остановится. Используйте кнопку «Остановить» для прерывания процесса.

## ⚙️ Настройки

Нажмите иконку **⚙️** в правом верхнем углу приложения:

| Параметр | Описание |
|----------|----------|
| **Папка** | Путь к папке с файлами комментариев |
| **VK API версия** | Версия VK API (по умолчанию `5.199`) |
| **Задержка** | Пауза между запросами к API (в секундах) |
| **API URL** | URL метода VK API |

Настройки сохраняются в `config.py`. При закрытии приложения файл сбрасывается к значениям по умолчанию.

## 🔐 Безопасность

- Токен **не сохраняется** на устройство — хранится только в памяти во время работы
- `config.py` **сбрасывается** к значениям по умолчанию при закрытии приложения
- Никогда не публикуйте и не передавайте свой токен
